AuthenticationRequest Up

identifier
String User identifier which can be either username, email, or phone number
password
String User's password for authentication

AuthenticationResponse Up

jwt (optional)
String JWT token generated after successful authentication
message (optional)
String Message indicating the result of the authentication process

OrderDto Up

id (optional)
Long Unique identifier for the order format: int64
orderDate
Date Date and time when the order was placed format: date-time
totalAmount
BigDecimal Total amount for the order
productIds
array[Long] List of product IDs included in the order format: int64
userId
Long Unique identifier of the user who placed the order format: int64

ProductDto Up

id (optional)
Long Unique identifier for the product format: int64
name
String Name of the product
description (optional)
String Description of the product
price
BigDecimal Price of the product
stockQuantity
Integer Quantity of the product in stock format: int32
supplierId
Long Unique identifier of the supplier associated with this product format: int64

SupplierDto Up

id (optional)
Long Unique identifier for the supplier format: int64
name
String Name of the supplier
contactNumber
String Contact number of the supplier
address
String Physical address of the supplier

UserDto Up

id (optional)
Long Unique identifier for the user format: int64
username
String Username for the user
password
String Password for the user account (hashed)
email
String Email address for the user
fullName
String Full name of the user
roles
array[String] Roles assigned to the user
dateOfBirth
date Date of birth of the user in dd-MM-yyyy format format: date
phoneNumber (optional)
String Phone number of the user
address (optional)
String Address of the user
isActive (optional)
Boolean Indicates if the user's account is active
